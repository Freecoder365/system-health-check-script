# System Health Check Script

## Description
This project is a Bash-based system health monitoring script that checks key system metrics such as memory usage, disk usage, system uptime, and running processes.

It helps monitor system performance and detect potential issues early.

---

## Features
- Checks available memory
- Displays disk usage
- Shows system uptime
- Lists top CPU-consuming processes
- Alerts when memory is below threshold

---

## Technologies Used
- Bash (Shell Scripting)
- Linux Commands: `free`, `df`, `ps`, `uptime`, `awk`
